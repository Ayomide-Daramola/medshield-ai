# MedShield AI

AI-powered fake medicine detection and verification platform for Nigeria.

## Features
- Medicine verification
- NAFDAC checker
- Barcode scanner
- Fake medicine alerts
- Pharmacy verification

## Tech Stack
- React
- Node.js
- Python
- TensorFlow
- PostgreSQL

## Project Goal
Help Nigerians identify fake and unsafe medicines using AI and public pharmaceutical data.

## Project Structure
- frontend/
- backend/
- ai-model/
- database/
- docs/

## Future Features
- AI image recognition
- Pharmacy locator
- Mobile app support
- Medicine expiry alerts
- QR code verification
